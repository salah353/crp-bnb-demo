# CRP BNB Demo

CRP BNB Demo is a minimal public demonstration of the CRP non-custodial payment infrastructure working on **BNB Chain**.

This demo shows a full transaction lifecycle:

1. Transaction prepared via CRP API
2. Transaction signed locally using CRP SDK
3. Transaction broadcast to BNB Chain
4. Transaction confirmed on BNB mainnet

This demonstrates how developers can integrate secure, non-custodial crypto payment flows into their applications.
